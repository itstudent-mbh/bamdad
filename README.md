This is a bamdad test frontend project with costomize tab and vertical slider

## Getting Started

First, run 
```bash
 npm install 
```
then run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
